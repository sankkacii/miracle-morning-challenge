같은 자리 같으면 0, 다르면 1 -> 딱봐도 XOR 연산

```python
class Solution:
    def hammingDistance(self, x: int, y: int) -> int:
        bin_string = bin(x ^ y)[2:]
        dist = 0
        for s in bin_string:
            if s == "1":
                dist += 1
        return dist
```
