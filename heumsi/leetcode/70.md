그냥 바로 DP로 풀면된다. 넘나 유명한 입문 문제임

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        d = [1, 2]
        for i in range(2, n):
            d.append(d[i-1] + d[i-2])
        return d[n-1]
```

위보다 좀 더 메모리 세이프하게 짤 수 있음.

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n == 1:
            return 1
        if n == 2:
            return 2
        first, second = 1, 2

        for i in range(3, n+1):
            answer = first + second
            first = second
            second = answer
        return answer
```
